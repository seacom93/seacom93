### Hi, I'm Sean

Database & Site Reliability Engineer. I like database internals — storage
engines, file formats, replication, and what actually happens between a query
arriving and bytes hitting disk.

**Day job:** running ClickHouse, PostgreSQL (Patroni), and Kafka pipelines on
Kubernetes in production. On-call, incident response, blameless post-mortems,
chaos testing, and building the tooling in Go that keeps it all observable.

**Here on GitHub:**

- [minimalparquet](https://github.com/seacom93/minimalparquet) — a pure-Go
  Parquet writer with Modular Encryption, built from the spec with zero
  dependencies. Made for encrypting event batches at the ingestion layer.
- [LumberJack](https://github.com/seacom93/LumberJack) — a SQL Server
  transaction log viewer in Go, because reading the WAL is the best way to
  understand a database.

**Interests:** ClickHouse and MergeTree internals, distributed consensus, CDC
pipelines, Kubernetes, and making reliability proactive instead of reactive.
