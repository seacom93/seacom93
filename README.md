### Hi, I'm Sean 👋

Database & Site Reliability Engineer. I like database internals: storage
engines, file formats, replication, and what actually happens between a query
arriving and bytes hitting disk.

**Day job:** running ClickHouse, PostgreSQL (Patroni) and Kafka pipelines on
Kubernetes in production. On-call, incident response, blameless post-mortems,
chaos testing, and building the Go tooling that keeps it all observable.

**Here on GitHub:** 🛠️

- [minimalparquet](https://github.com/seacom93/minimalparquet) is a pure Go

  dependencies. I built it for encrypting event batches at the ingestion
  layer.
- [LumberJack](https://github.com/seacom93/LumberJack) is a SQL Server
  transaction log viewer in Go. Reading the tlog is the best way to understand
  a database.

**Interests:** ClickHouse MergeTree internals, distributed consensus, CDC
pipelines, Kubernetes, and making reliability proactive instead of reactive.
